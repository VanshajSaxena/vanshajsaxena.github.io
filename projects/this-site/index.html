Source Code

This site is built with Hugo.

This allowed me to showcase my projects and my thoughts to you. The site has a really simple hierarchical structure, supporting blogs and project posts.

Writing a simple blog post#

Writing a simple blog post like this involves simply creating a markdown file with specific archetype in the posts directory. Or you can use Hugo binary to create a new post.

vanshaj@NIXOS ~/ (master+*) $ hugo new content /path/to/post

Then simply writing the post is all you need to do.
